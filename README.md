# Django Word Counter

This is a simple web application built using Django that counts the number of words entered by the user and displays word statistics.

## Features
- Accepts text input from users
- Counts total number of words
- Displays processed results on a separate page

## Tech Stack
Python  
Django  
HTML  
CSS  

## Project Structure
manage.py  
views.py  
urls.py  
templates (HTML pages)

## How to Run the Project

1. Clone the repository
git clone https://github.com/Sheshumadhava/django-word-counter

2. Navigate to the project folder
cd django-word-counter

3. Install Django
pip install django

4. Run the server
python manage.py runserver

5. Open in browser
http://127.0.0.1:8000
